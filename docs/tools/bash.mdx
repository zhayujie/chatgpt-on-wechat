# bash

在当前工作目录执行 Bash 命令，返回 stdout 和 stderr。`env_config` 中配置的 API Key 会自动注入到环境变量中。

## 依赖

无额外依赖，默认可用。

## 参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `command` | string | 是 | 要执行的命令 |
| `timeout` | integer | 否 | 超时时间（秒） |

## 使用场景

- 安装软件包和依赖
- 运行代码和测试
- 部署应用和服务（Nginx 配置、进程管理等）
- 系统运维和排查
