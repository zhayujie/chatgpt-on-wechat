# read

读取文件内容。支持文本文件、PDF 文件、图片（返回元数据）等格式。

## 依赖

无额外依赖，默认可用。

## 参数

| 参数 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- |
| `path` | string | 是 | 文件路径，相对路径基于工作空间目录 |
| `offset` | integer | 否 | 起始行号（1-indexed），负值表示从末尾读取 |
| `limit` | integer | 否 | 读取行数 |

## 使用场景

- 查看配置文件、日志文件
- 读取代码文件进行分析
- 检查图片/视频的文件信息
